0. "python" on its own on the terminal enters a REPL
1. "ipython" on its own on the terminal enters a REPL w/ more bells + whistels
2. "python filename.py" is how run a specific python script from the command line.
3. "jupyter notebook" runs the jupyter notebook server in your pwd.
4. "jupyter notebook notebook_name.ipynb" to run a specific notebook on the server
